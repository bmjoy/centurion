Class CClassFactory
This class is used to create classes that wrapper Elements of the XML Schema.
  Members Description  
    ParamsLib_Classes The enumerations for each of the classes

  
    CreateClass Creates an instance of an object (Static)

  
    CreateClassCollection Creates an instance of a class collection (Static)

  
    CreateEnumCollection Creates an instance of an enumeration collection (Static)

  
    FromXml Creates a suitable wrapper class and loads an XML string into the class. (Static)

  
    FromXmlFile Creates a suitable wrapper class and loads an XML file into the class. (Static)

  
    FromXmlStream Creates a suitable wrapper class and loads XML data from a CBinaryData object into the class. (Static)

  
    FromXmlElement Creates a suitable wrapper class and loads data from an XML element into the class. (Static)
