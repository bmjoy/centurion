LtXmlLib18::CBigInteger GetYPosition() const;
void SetYPosition(LtXmlLib18::CBigInteger val);
  Property Description  
    Property Name yPosition  
    Property Type LtXmlLib18::CBigInteger  
    Parent Class ParamsLib::CCamera  
    XML Type Element  
    Accessors Read/Write  
    Description A mandatory CBigInteger  
    Remarks It is mandatory and therefore must be populated within the XML.
It is defaulted to [0].

 
    Example 
ParamsLib::CCameraPtr spParent = ...;

// Because the property is mandatory, it will already be populated

// So it can be read  as follows (Note: Items in the collection are held by value)
printf(_T("Object Value (%s)\n"), spParent->GetYPosition()->ToString().c_str());

// So it can be set as follows 
spParent->GetYPosition().Parse(_T("45578621856"));

// Or like this
LtXmlLib18::CBigInteger newItem;
newItem.Parse(_T("45578621856"));
spParent->SetYPosition(newItem);
