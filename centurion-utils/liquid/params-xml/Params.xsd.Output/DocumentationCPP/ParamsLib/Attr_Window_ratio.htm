LtXmlLib18::CDecimal GetRatio() const;
void SetRatio(LtXmlLib18::CDecimal val);
  Property Description  
    Property Name ratio  
    Property Type LtXmlLib18::CDecimal  
    Parent Class ParamsLib::CWindow  
    XML Type Element  
    Accessors Read/Write  
    Description A mandatory CDecimal  
    Remarks It is mandatory and therefore must be populated within the XML.
It is defaulted to [0].

 
    Example 
ParamsLib::CWindowPtr spParent = ...;

// Because the property is mandatory, it will already be populated

// So it can be read  as follows (Note: Items in the collection are held by value)
printf(_T("Object Value (%s)\n"), spParent->GetRatio()->ToString().c_str());

// So it can be set as follows 
spParent->GetRatio().Parse(_T("45578621856.4546465"));

// Or like this
LtXmlLib18::CDecimal newItem;
newItem.Parse(_T("45578621856.4546465"));
spParent->SetRatio(newItem);
