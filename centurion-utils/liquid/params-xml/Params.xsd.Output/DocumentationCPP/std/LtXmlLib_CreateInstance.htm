static CXXXXPtr CreateInstance();
  Property Description  
    Method Name CreateInstance  
    Returns A newly created instance of the object.  
    Description Creates a new object.  
    Remarks This is the recomended way to create new objects

The object returned is wrapped inside a smart pointer.
This takes care of all the reference counting issues.

The returned object is initilized. All the mandatory attributes and elements will have been created, optional attributes and elements are never created, and collections are always created empty. If a collection is required to have 1 or more items in it then validation may fail when calling ToXml (until you add the appropriate number of items to the collection).
